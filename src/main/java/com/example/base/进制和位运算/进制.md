原码反码补码
编码规则：
有符号数的编码规则：

原码：最高位符号位，对其他的位进行本身的绝对值即可
反码：
    正数：反码和原码相同
    负数：符号位一定是1，其余位对原码取反。
补码：    
    正数：补码和原码相同
    负数：符号位一定是1，然后对反码+1
